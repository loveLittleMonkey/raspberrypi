# 🔥录系统

```shell
// 查看盘符

df -lh

// 写在sd卡

sudo diskutil umount /dev/disk2s1

// 再次确认盘符

df -lh

// 开始烧录系统（成功后重新插卡，有boot盘）

dd bs=1m if=./2020-05-27-raspios-buster-full-armhf.img of=/dev/rdisk2

// 在sd卡中设置空白的ssh文件，开启ssh链接

cd /Volumes/boot/ && touch ssh

// 直连macbook pro后查看与本机相关的ip，便于ssh链接

arp -a
```
